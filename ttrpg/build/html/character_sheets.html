Magic Reference
Casting Loop:
1. Channel: Roll Wits + Arcana
2. Each Success = 1 Potential
3. Each 1 = 1 CP immediately
4. Weave: Roll Wits + (Art)

6. Backlash CP spent by GM

• 5+ CP: Major turn
