\subsection*{The 10% Rule}

From \autoref{ch:gm‚ąícurrency‚ąífree} (and echoed in \autoref{ch:gm‚ąíreference}): pick \textit{one} subsystem from this book and use it tonight. Next session, pick another. The game will not collapse if you never use the Deck of Consequences (\autoref{app:deck‚ąímagic}) or the travel timers (\autoref{sec:gm‚ąítravel}). The game \textit{will} collapse if you try to track every timer, every faction, every threshold effect, and every player‚ąímanaged module at once.

\subsection*{What This Book Is Not}

It is not a prescribed curriculum. It is not a contract. It is a set of lenses ‚ąí‚ąí‚ąí put them on when you need focus, take them off when they blur the story. The Golden Rule (see \autoref{ch:gm‚ąíintro}) applies to the GM as much as to the players: when in doubt, make a ruling that keeps the story moving. Default to DV 3, Position Controlled, and let the dice fall.
