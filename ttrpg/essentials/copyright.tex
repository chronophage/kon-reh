  {\Large\textbf{System Reference Document (SRD) Designation}}\\[0.3in]

  \rule{0.7\textwidth}{0.4pt}\\[0.3in]

  \begin{minipage}{0.85\textwidth}
    \small
    This document contains material designated as the \textbf{Fate's Edge System Reference Document (SRD)}.
    The SRD is the open-license portion of this work and is intended for use by third-party creators,
    developers, and community members.\\[0.2in]

    \textbf{SRD Content Includes:}
    \begin{itemize}
      \item Core resolution mechanics (dice pools, Position, Difficulty Value, Outcome Matrix)
      \item Boon and Story Beat economies
      \item Fatigue, Harm, and Armor Conversion systems
      \item Attribute and Skill descriptions
      \item Basic combat and action economy rules
      \item Universal magic mechanics (TAGS, DV calculation, Backlash)
      \item NPC templates and encounter guidelines
      \item Equipment and gear tables
      \item Character creation framework
      \item Optional systems (Deck of Consequences, Crown Spread, Travel framework)
    \end{itemize}

    \vspace{0.2in}

    \textbf{Extended Content (Not SRD):}
    \begin{itemize}
      \item Setting lore, regional descriptions, and cultural details
      \item Original characters, NPCs, and named figures
      \item Adventure modules and scenario hooks
      \item Patron descriptions, Rites, and exclusive mechanical content
      \item Proprietary magic systems (Runekeeper, Invoker, Cantor, Summoner-specific expansions)
      \item Artwork, maps, and graphical elements
      \item Faction descriptions and campaign-specific content
      \item Original prose, framing devices, and narrative text
    \end{itemize}
