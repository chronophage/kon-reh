*5+ = Extreme / Set-Piece Challenge*

**Default by Position:**  
*Dominant = DV 2*

## ‚ö° Story Beats (SB)

*Generated by player rolls (1s)*  
*Spent by GM to drive the fiction*

**Spend Menu:**  
- 1 SB = Minor pressure (alarm, time loss, supply tick)  
- 2 SB = Moderate setback (lose cover, small foe arrives)  
- 3 SB = Major trouble (reinforcements, key gear breaks)  
- 4+ SB = Dramatic shift (trap, authority arrives, scene changes)

## üé≠ Boons (Player Currency)

## üÉè Deck of Consequences

**Use a standard 52-card deck when SB hits hard.**

**Suits = Type of Complication:**  
- Spades: Harm / Danger  
- Hearts: Social / Reputation  
- Clubs: Resources / Fatigue  
- Diamonds: Arcane / Mystical  

**When to Draw:**  
- Desperate success with cost

**[TAGS Tip]:** Use suit meanings to guide how [TAGS] complications manifest

## üìã GM Flow (Scene Procedure)

1. **Frame stakes:** Position + Effect + Cost of failure  
2. **Set DV:** Based on Position + fiction (add +1 for each TAG used)  
3. **Roll:** Attribute + Skill pool  
4. **Count:** 6+ successes, 10s double, 1s = SB  
5. **Resolve:** Apply Outcome Matrix  
6. **Spend SB:** Complications, Deck draws, clock ticks  
7. **Shift Position:** Fictional consequences or Boons  

**[TAGS Note]:** For magic, note relevant TAGS when setting DV

## üïí Clocks

**Use to track:** enemy waves, collapsing ruins, social resistance, supply, corruption.  
**[TAGS Note]:** Certain TAGS (like [WARD] or [BANISH]) can advance or hold clocks

## üîñ [TAGS] Quick Reference

**Core System:** Narrative shorthand for magical effects

**How to Use:**  
- Describe effect ‚Üí Assign TAGS ‚Üí Set DV (1 + number of TAGS)

- [DISPEL] [COUNTER] (Counter-magic)  
- [SPEED] [STABILITY] (Movement)  
- [HARM] [HEAL] (Effect type)

**Backlash Rule:** On mixed results, Backlash triggers with TAGS coloration
