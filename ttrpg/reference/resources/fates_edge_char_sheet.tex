% ----- Intro Page -----
\section*{How to Use This Booklet}
Print the pages you need. Start with the \textbf{Core Character Sheet}.\\
Then choose the module that matches your character’s magical path (Runekeeper, Invoker, Cantor, Summoner).\\
If you have Followers or Assets, print the \textbf{Followers \& Assets} page as well.

\vspace{0.5cm}
\noindent All sheets are designed to be printed on standard letter paper (8.5 × 11″). Use a pencil or a dry‑erase marker if you laminate them.
