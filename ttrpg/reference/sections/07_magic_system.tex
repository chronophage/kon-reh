\subsection{Rites Difficulty Value (Expanded)}Ňāabel{sec:rites_dv‚āĎxpanded}
‚ąądex{Rites!Difficulty Value}‚ąądex{Difficulty Value (DV)!Rites}

The Difficulty Value (DV) to cast a Rite is:
\[
\text{DV} = \max\!\bigl(\text{Obligation Cost} ‚ąí \text{Spirit},‚Äā \text{Tier}\bigr)
\]

\begin{description}
    ‚ĄĻtem[Obligation Cost:] The Rite's listed cost in Obligation segments.
    ‚ĄĻtem[Spirit:] The caster's Spirit attribute. Each point reduces the effective weight of the Obligation.
    ‚ĄĻtem[Tier:] The Rite's intrinsic difficulty based on scope or potency. DV can never fall below this floor.
\end{description}

\paragraph{Worked Examples}
\begin{itemize}
    ‚ĄĻtem \emph{Novice:} Tier~1 Rite, Obligation Cost~2, Spirit~1 ‚Üí DV = $\max(2‚ąí1,1)=1$.
    ‚ĄĻtem \emph{Mid‚ÄĎlevel:} Tier~2 Rite, Obligation Cost~4, Spirit~2 ‚Üí DV = $\max(4‚ąí2,2)=2$.
    ‚ĄĻtem \emph{Advanced:} Tier~3 Rite, Obligation Cost~7, Spirit~3 ‚Üí DV = $\max(7‚ąí3,3)=4$.
\end{itemize}

\subsection{Obligation Capacity}Ňāabel{sec:obligation_capacity}
‚ąądex{Obligation!Capacity}

\begin{itemize}
    ‚ĄĻtem \textbf{Exceeding Capacity:} For each segment above Capacity, mark 1 Fatigue. The character cannot Invoke Rites or perform rituals until Obligation is reduced below Capacity.‚ąądex{Fatigue}
    ‚ĄĻtem \textbf{Resolution:} Reduce Obligation through Downtime service, Patron tasks, ritual cleansing, or story resolution.‚ąądex{Downtime}
\end{itemize}
\textit{Example:} Spirit~2 + Presence~3 = Capacity 5. 6 segments ‚Üí Fatigue 1; 7 segments ‚Üí Fatigue 2; 10 segments ‚Üí Harm 1; 11 segments ‚Üí Harm 2.
