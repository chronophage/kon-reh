\subsection*{A Reading Plan for New Players}

You do not need to read 200 pages to play your first session. Here is what you actually need:

\begin{enumerate}
    ‚ĄĻtem Read \autoref{ch:intro} ‚Äď especially the safety tools (Lines, Veils, X‚ÄĎCard) and the Session Zero checklist.
    ‚ĄĻtem Read \autoref{ch:core‚ąímechanics} up to \autoref{sec:boons} ‚Äď the play loop, Position, Difficulty, and the Outcome Matrix.
    ‚ĄĻtem Skim \autoref{sec:harm‚ąífatigue} to understand how Harm and Fatigue work. You will refer back to it.
    ‚ĄĻtem Use the character creation checklist (\autoref{ch:character‚ąícreation}) to build your first character. The pre‚ÄĎgenerated examples in \autoref{ch:example‚ąíconcepts} can speed this up.

    ‚ĄĻtem Print the one‚ÄĎpage player reference (if your GM provides it) or copy the Outcome Matrix onto an index card.
    ‚ĄĻtem Close the book and play.
\end{enumerate}

\subsection*{A Note on the Quickstart}

If you are completely new, start with the \textit{Fate's Edge Quickstart}. It contains the core rules, four pre‚ÄĎgenerated characters, and a complete adventure (``The Lantern at Dusk''). Run that first. Then open this book to build your own character for the \textit{Blood and Silk} trilogy.

\subsection*{The Golden Rule (for Players)}

\begin{quote}
``When in doubt, make the choice that serves the story. If you find yourself asking `What's the optimal move?' try reframing it as `What would be most interesting for the story?'''
\end{quote}

That permission is written into the rules. Use it.
