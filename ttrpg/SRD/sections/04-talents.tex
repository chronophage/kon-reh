\section{Skills, Talents, and Equipment}

\subsection{Core Skills (Condensed)}
An action roll pairs an \textbf{Attribute} with a \textbf{Skill}. The GM sets Position and DV from the fiction. Core skills include:

\begin{itemize}[leftmargin=*]
  \item \textbf{Combat:} Melee, Ranged, Brawl, Tactics.
  \item \textbf{Physical:} Athletics, Stealth, Endurance, Craft, Survival.
  \item \textbf{Social:} Sway, Command, Deception, Performance, Insight.
  \item \textbf{Knowledge:} Lore, Investigation, Medicine.
  \item \textbf{Magic:} Arcana, Ritual.
\end{itemize}

\begin{itemize}[leftmargin=*]
  \item \textbf{Arcana} -- magic theory, wards, free casting, counter-spelling.
  \item \textbf{Athletics} -- running, climbing, jumping, swimming.
  \item \textbf{Brawl} -- unarmed combat, grappling, improvised weapons.
  \item \textbf{Command} -- leadership, intimidation, issuing orders.
  \item \textbf{Craft} -- building, repairing, creating tools or structures.
  \item \textbf{Deception} -- lying, bluffing, misdirection, forgery.
  \item \textbf{Endurance} -- resisting fatigue, poison, harsh weather.
  \item \textbf{Insight} -- reading emotions, detecting lies, understanding motives.
  \item \textbf{Investigation} -- research, deduction, analysis of clues.
  \item \textbf{Lore} -- history, culture, religion, customs, local knowledge.
  \item \textbf{Medicine} -- healing, anatomy, first aid, surgery.
  \item \textbf{Melee} -- swords, axes, close-quarters weapons.
  \item \textbf{Performance} -- singing, acting, oration, musical display.
  \item \textbf{Ranged} -- bows, crossbows, thrown weapons.
  \item \textbf{Ritual} -- performing rites, invocations, exorcisms, consecrations.
  \item \textbf{Stealth} -- moving unseen, hiding, camouflage.
  \item \textbf{Survival} -- tracking, foraging, predicting weather, navigating wilderness, animal handling.
  \item \textbf{Sway} -- persuasion, negotiation, charm.
  \item \textbf{Tactics} -- battlefield strategy, unit coordination, ambush planning.
\end{itemize}

\noindent
\textit{Note:} \textbf{Survival} replaces the old \textit{Nature} skill, covering all wilderness and animal interactions. \textbf{Insight} replaces \textit{Notice} for social perception; use \textbf{Investigation} for physical clue-finding. \textbf{Ritual} is the skill for actively performing magic (Rites, Songs, invocations); \textbf{Arcana} covers theory and identification.  
Optional modules (e.g., \textit{Psionics}) may add additional skills.

\noindent
\textbf{Common uses:} Use \textbf{Craft} for mechanical locks and traps; \textbf{Arcana} or \textbf{Lore} for magical wards. \textbf{Sway} gets you past people; \textbf{Stealth} and \textbf{Craft} get you through mechanisms.
\subsection{Talents}

Talents are the building blocks of character specialization. Each costs XP. \textbf{Passive} talents are always on; \textbf{active} talents require an action or reaction and cannot be used simultaneously unless noted.

\paragraph{Talent Costs}
\begin{itemize}
  \item 2--3 XP: Minor edge (small bonus, once per scene).
  \item 4--6 XP: Major edge (permanent +1 die in a niche, access to magic).
  \item 7--10 XP: Prestige (campaignâ€‘defining).
  \item 11+ XP: Epic (legendary, storyâ€‘shaping).

\subsection{Equipment Condition \& Upkeep}

All gear can become \textbf{Neglected} or \textbf{Compromised}:
\begin{itemize}
  \item \textbf{Neglected}: Weapons --1 die; Armor conversion worsened by 1 step.
  \item \textbf{Compromised}: Weapons --1 die first attack/round; Armor provides no conversion.
\end{itemize}
\textbf{Repair:}
\begin{itemize}
  \item Short rest + tools: Remove Neglected.
  \item A scene of work or a smith: Remove Compromised.
  \item Enchanted items require the same repairs; enchantments are inactive while Compromised.
\end{itemize}

\subsection{Magical Items (Enchantments)}

Enchantments are permanent magical modifications treated as Talents (XP cost). They follow the same stacking rules: only one active enchantment per item unless noted. Enchanted items count as Assets for upkeep.

\subsubsection*{Sample Enchantments}
\begin{itemize}
  \item \textbf{Keen Edge (2 XP)}: +1 die vs armored targets.
  \item \textbf{Flaming Blade (4 XP)}: +1 fire damage, +1 Effect vs cold creatures.
  \item \textbf{Soulfire Weapon (6 XP)}: Ignores 1 point of armor/resistance.
  \item \textbf{Shadowweave Armor (2 XP)}: +1 die to Stealth while moving.
  \item \textbf{Runed Plate (4 XP)}: Reduce magical Backlash by 1 SB.
  \item \textbf{Boots of Sure Step (3 XP)}: Ignore difficult terrain.
\end{itemize}

\subsubsection*{Enchantment Limits}
\begin{itemize}
  \item Max enchantments = item's Tier (Light=1, Medium=2, Heavy=3).
  \item Identical benefits do not stack.
  \item Removing an enchantment: ritual (Wits+Arcana DV 4) + 1 XP per enchantment level.
\end{itemize}
